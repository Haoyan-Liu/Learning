# 树遍历与二叉树性质

## 树的遍历方法

- **前序遍历（Pre-order）**：根 → 左 → 右
- **中序遍历（In-order）**：左 → 根 → 右
- **后序遍历（Post-order）**：左 → 右 → 根

## 平衡二叉树（Balanced Binary Tree）

每个节点的左子树和右子树高度差不超过 1。

## 完全二叉树（Complete Binary Tree）

所有层都被填满，最后一层的节点尽可能靠左。

**完全二叉树（CBT）一定是平衡二叉树。**

## 二叉搜索树（BST）

- **左子树规则**：左子节点的值小于父节点。
- **右子树规则**：右子节点的值大于父节点。
- **通常不允许重复值。**
